# Development notes

## Versioning
- Bump `@version` in `autodarts-core.user.js`
- Add entry to `CHANGELOG.md`

## Publishing
- Commit changes to `main`
- Users can update via Violentmonkey check-for-updates

## Tips
- Keep selectors stable where possible
- Prefer `#ad-ext-turn`, `#ad-ext-player-display`, and custom classes over Chakra hashed classes
